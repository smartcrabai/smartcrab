# AGENTS.md

## Language

All natural-language text in this repository must be written in **English**. This includes:

- Commit messages
- Code comments
- Documentation (READMEs, design docs, etc.)
- Pull request titles and descriptions
- Issue titles and descriptions
- Log messages and error messages in code
- Any other prose intended for humans

Code identifiers and technical terms should follow standard conventions and remain as-is.
